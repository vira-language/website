About Vira Lang

Vira Lang is a compiled programming language with syntax identical to C, making it easy for C developers to transition. It leverages modern tools for compilation, error diagnostics, and package management.

Core Components

  • Compiler: Built with Rust and Cranelift, compiles .vira files to object files.
  • Preprocessor: Handles #include, #define, etc., similar to C.
  • Parser & Semantic Checker (PLSA): Written in C++, ensures syntax and semantics are correct.
  • Diagnostic Tool: Uses miette for beautiful error reporting.
  • CLI Tools:
    • vira: General CLI for compiling and updating.
    • virac: Dedicated compilation tool.
    • virus: Package manager inspired by Cargo, uses Podman for isolated builds with Wolfi base image.
  • Updater: Checks GitHub for new versions and downloads updates.

Syntax Example

int main() {
    return 0;
}

Vira Lang supports basic C constructs like functions, returns, binary operations, and more, with plans for full C compatibility.
