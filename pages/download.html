Installation Instructions

  1. Download the ZIP file for your OS.
  2. Extract to the appropriate directory (e.g., /usr/lib/vira-lang on Linux, C:\Program Files\ViraLang on Windows).
  3. Add the bin directory to your PATH.
  4. Run vira update to ensure everything is up to date.
