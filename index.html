Welcome to Vira Lang

A modern, efficient programming language with syntax identical to C, designed for high performance and ease of use.

Key Features

  • C-like syntax for familiar development
  • Compiled to native code using Cranelift
  • Built-in preprocessor, parser, and semantic checker
  • Package manager (virus) with Podman integration
  • Cross-platform support (Linux, Windows)
  • Automatic updates and diagnostic tools
